# Chapter 8: MDL Across the Stack

The amended claim: MDL applies universally but to different objects. Neural components minimize description length of information and behavior. Cryptographic components minimize description length of trust and provenance. A Merkle tree doesn't compress data — it compresses verification (O(N) to O(log N)). Every layer compresses; they optimize against different physical limits. Show this isn't a loose analogy by walking through at least three concrete layers.
