# Gemini Review — 2026-03-16

## Key Takeaways

1. **"You Are Here" pointer** — Agent needs 5 sequential reads to find work. Add explicit active milestone to ROADMAP.md.
2. **Archive completed work** — Keep active directories sparse. Move done stories to `archive/`.
3. **Shard suggestions** — Individual files in `suggestions/` instead of one growing list in ROADMAP.md.
4. **CLI aggregators** — `ls`/`cat` fail at cross-file queries ("all pending tasks", "what blocks X"). Provide `bin/plan-*` scripts.
5. **JSONL fragility** — LLMs mutating JSON inside markdown code blocks is error-prone. Either use markdown checklists or provide a `bin/update-task` tool.
6. **State desync** — Agent can mark tasks done but forget to update milestone checklist. Need a consistency check.
7. **Concurrent writes** — Web agent and coding agent editing same file = conflicts. Suggestions should be individual files.
